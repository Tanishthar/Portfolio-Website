Description

A chatbot designed for answering student queries related to course details, faculty information, exam schedules, and department policies. Uses NLP and a knowledge base to provide instant responses and reduce faculty workload.

Tech Stack

  • React: A JavaScript library for building dynamic user interfaces; it powers the frontend of your project, enabling interactive and responsive web pages.
  • Tailwind CSS: A utility-first CSS framework used for styling; it helps you quickly design clean and customizable UI components in your React frontend.
  • Django: A high-level Python web framework used to handle backend logic, routing, and database management in your project.
  • Python: The core programming language used for backend development, data processing, and API integration (like Gemini API) in your project.
  • Gemini API: A language model API by Google that powers conversational or generative AI features in your project, such as answering queries or generating text.
