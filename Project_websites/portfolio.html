Description

This is a personal portfolio website designed to showcase my work, interests, and journey. It features a sleek, dark-themed interface with a dynamic starfield background and interactive 3D elements. The site highlights my projects through standalone pages with video demos, descriptions, and GitHub links. A rotating 3D model introduces the site and doubles as an AI-powered chatbot for personalized interaction. Additional sections include an about me page, a gallery of my travels and hobbies, and a contact area for collaboration or networking.

Tech Stack

  • HTML5 & CSS3 – Structure and styling of all web pages
  • Three.js – For rendering and animating the interactive 3D model
  • JavaScript – Interactive features
