Bicep Curl Rep Counter
Using Computer Vision

↓ Scroll Down ↓

Description

The Bicep Curl Tracker is an AI-powered fitness application that uses computer vision to count repetitions and analyze form during bicep curls. By leveraging pose detection, it tracks key joint angles (shoulder, elbow, wrist) to identify the upward and downward motion of a bicep curl. It provides instant feedback, rep count, and can optionally log workout sessions for progress tracking. Ideal for home workouts, fitness training, or physical therapy.

Key Features:

  • Real-time bicep curl detection using webcam.
  • Accurate rep counting based on elbow angle.
  • Visual feedback with skeletal overlay.

    • Tech Stack

  • Language: Python
  • Computer Vision: OpenCV
  • Pose Estimation: Mediapipe Pose
  • Mathematics: NumPy (for angle calculations)
  • GUI: OpenCV overlays
