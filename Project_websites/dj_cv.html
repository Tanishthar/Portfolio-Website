DJ Control

Description

This project enables users to control a DJ system using hand gestures. It uses computer vision to detect hand movements and adjusts volume and speed accordingly.

Tech Stack

  • Python (OpenCV, MediaPipe)
  • SoundDevice & SoundFile for audio control
  • Machine Learning for gesture recognition
